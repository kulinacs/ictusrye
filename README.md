# Ictusrye

A dark bootstrap theme for Jekyll

## Installation

Add this line to your Jekyll site's Gemfile:

```ruby
gem "ictusrye"
```

And add this line to your Jekyll site's `_config.yml`:

```yaml
theme: ictusrye
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ictusrye

## Usage
